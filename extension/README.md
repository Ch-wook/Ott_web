# Extension

OTT 페이지에서 현재 시청 중인 콘텐츠 정보를 추출하기 위한 크롬 확장프로그램입니다.

## 소개
이 확장프로그램은 OTT 사이트에서 사용자가 보고 있는 콘텐츠의 정보를 확인하고,
팝업 화면에서 이를 볼 수 있도록 구성되어 있습니다.

현재는 페이지 정보 추출 기능을 중심으로 구현되어 있으며,
추후 다른 플랫폼 지원 및 대시보드 연동 기능을 확장할 예정입니다.

## 주요 기능
- 현재 페이지의 OTT 정보 확인
- 콘텐츠 제목 추출
- 시청 진행률 확인
- 팝업 창에서 추출 결과 출력

## 파일 설명
- `manifest.json` : 확장프로그램 설정 파일
- `popup.html` : 팝업 UI
- `popup.js` : 팝업 동작 및 정보 추출 실행
- `disney-bridge.js` : 디즈니플러스 페이지 정보 추출 스크립트

## 실행 방법
1. 크롬 브라우저에서 `chrome://extensions` 접속
2. 우측 상단의 개발자 모드 활성화
3. `압축해제된 확장 프로그램 로드` 클릭
4. `extension` 폴더 선택
5. OTT 페이지에서 확장프로그램 실행 후 정보 확인

## 주의사항
- OTT 사이트 구조가 변경되면 일부 추출 기능이 정상 동작하지 않을 수 있습니다.
- 현재 구현은 확장 및 테스트 단계에 있으며, 추후 지원 플랫폼이 추가될 수 있습니다.
